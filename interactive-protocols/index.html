Protocols

  • Genomic DNA Extraction (interactive)

How to use in Notion

  1. Enable GitHub Pages in Settings → Pages and copy your site URL.
  2. In Notion, type /embed and paste the link to any protocol page (e.g., .../protocols/dna-extraction/gdna.html).
  3. Resize the embed block as needed. Checkboxes autosave per browser using localStorage.
